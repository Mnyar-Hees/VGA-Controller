# VGA 640x480 @ 60 Hz FPGA Project

**Engineer:** Mnyar Hees  
**Target Device:** Intel/Altera MAX 10 (DE10-Lite board)  
**Tools Used:** Intel Quartus Prime & ModelSim - Intel FPGA Edition  

---

## Description

This project implements a VGA controller for 640x480 resolution at 60 Hz on the DE10-Lite FPGA board.  
It generates horizontal and vertical sync signals and allows simple RGB output controlled by push-buttons.

- **Pixel Clock:** 25 MHz (generated by dividing the 50 MHz system clock)  
- **Inputs:** 3 push-buttons (KEY0, KEY1, KEY2) to select Red, Green, or Blue  
- **Outputs:** VGA_HS, VGA_VS, VGA_R, VGA_G, VGA_B  

The design demonstrates:

- Synchronous reset handling with metastability protection  
- VGA timing generation  
- Simple color selection using buttons
